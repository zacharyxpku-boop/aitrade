# Course Completion Evidence

`buildLearnerProgressReport` returns `courseCompletionEvidence` for active learner course packages.

Each evidence record lists required education items across:

- knowledge review
- scenario practice
- backtest metric misconception drill
- market context misconception drill
- source transparency misconception drill

This evidence is read-only learning proof for coach/admin review and future education-model routing. It is not trading skill certification, a live signal, return evidence, or real-money readiness.

Admin progress reports also expose `activity.courseEvidenceBlockers` so operators can see why a learner cannot receive, or has already received, a completion report.
