# Education Tutoring Plan

`buildEducationTutoringPlan` turns the safe `educationModelContext` into a tutoring plan for learner support and human-coach review.

The plan includes:

- focus areas
- lesson steps
- coach questions
- next education-only practice
- moderation status
- prohibited uses inherited from the model context

The plan is generated from learning evidence only. It does not read raw K-line series, live market data, broker data, or real-money order data.

It must not be used for:

- stock recommendations
- live buy/sell signals
- return promises or estimates
- broker or auto-trading actions
- real-money trading instructions
