# 知识来源审查

本文件记录第一版知识工程系统的来源边界。所有外部来源只用于 taxonomy、分类、schema 和覆盖范围参考；学习者可见内容必须保持原创教育表达。

## 原则

- 不复制 GitHub、博客、课程、README、书籍或第三方内容。
- 不把开源技术分析库输出包装成交易信号。
- 不把 demo 图表案例包装成真实授权行情。
- 不把新闻、情绪或事件背景当作真实资金动作理由。
- 所有来源进入产品前都需要记录 license、可靠性等级、可用范围和不可用范围。

## 可靠性等级

- S：官方文档、交易所/监管/数据源文档。
- A：成熟开源库或自有原创教育框架。
- B：有参考价值的开源项目或公开教育结构。
- C：普通博客、论坛、个人经验。
- D：不可进入知识库，只能作为研究线索。

## 当前状态

第一版知识库是教育底座，不是商业化最终版。后续如果接入真实行情、新闻、情绪或截图识别，必须补充授权、展示权、缓存权、延迟/实时披露和输出审计。
