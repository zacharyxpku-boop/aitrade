## Completion Rule

Each blank-preview target needs one explicit reviewer decision, reviewer note, evidence path or unrecoverable rationale, and rerun evidence before source-fit and public-grounding gates can continue.

## Boundary

Source replacement decision worksheet is a reviewer-only control layer for blank-preview local private course PDFs. It does not replace files, infer missing content, approve learner-facing release, provide trading advice, stock recommendations, live signals, return promises, broker workflows, automation, or real-money guidance. Neighbor candidates may be used as context only after reviewer confirmation, not as direct replacements.
