# Knowledge Quality Gaps

The source pool has moved from a mostly generated structure toward a real metadata layer, but the product is still not a finished professional curriculum.

## Current Strengths

- Real source pool is now 11,221 URLs.
- Final-source search URL count is 0 in `docs/REAL_SOURCE_HARVEST.json`.
- The source pool now includes GitHub repositories, npm package pages, npm linked project URLs, official documentation, public education pages, exchange docs, macro data docs, and data-provider docs.
- Source records keep education-only boundaries and `productionReady:false`.

1. Official source depth is still low.
   - Current official/public document URLs: 841.

   - Public-document use tiers are now explicit: 270 green official/public-domain sources, 448 yellow metadata/citation-only sources, and 123 red terms-restricted references.
   - The next step is terms/access review depth on these sources, not raw URL count.

2. npm breadth is not the same as education quality.
   - npm pages and linked URLs are useful for taxonomy discovery.
   - They cannot be treated as learner-facing authority without license, maintenance, and relevance review.

2a. Authority tiers are now explicit, but still need review.
   - Tier S: 335.
   - Tier A: 62.
   - Tier B: 2,724.
   - Tier C: 47.
   - Research/review only: 8,053.
   - Tiering is metadata classification, not content reuse permission.

3. `research_only` remains high.
   - Current research-only sources: 8,052 after relevance review.
   - 103 sources are explicitly off-topic review-only.
   - 4,310 sources are weak-relevance review-only.
   - These must not enter learner-facing content until reviewed.
